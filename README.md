# ⚙️ 3D Mechanical Designs

## 📌 Overview
This repository contains a collection of 3D CAD models designed using Onshape as part of the Mechanical Engineering Training Program.

The project demonstrates 3D modeling skills, CAD design workflow, and digital model development with preparation for STL export and 3D fabrication.

## 🛠️ Designed Models

### 🦇 1. Batman Logo Model
- ✏️ Designed using Onshape CAD tools.
- 🖨️ Developed as a 3D printable model.
- 📐 Focused on surface modeling and geometric design concepts.

### ❤️ 2. Heart Shape Model
- 🔧 Created using parametric CAD modeling techniques.
- 📏 Designed to enhance skills in sketching, modeling, and feature development.

## 💻 Tools & Technologies
- 🖥️ CAD Software: Onshape
- 📂 File Format: STL
- ⚙️ Design Approach: 3D CAD Parametric Modeling

## 🚀 Skills Demonstrated
- 🧩 3D CAD Modeling
- ✏️ Sketch Creation and Feature Development
- 📐 Engineering Design Workflow
- 🖨️ STL File Preparation for Manufacturing Applications

## 🎓 Training Context
Developed as part of the Intelligent Methods / Mechanical Engineering Training Program to practice digital design and engineering modeling techniques.
